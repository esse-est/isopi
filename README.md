# Isopi

Isopi is a python script that's created for the management and reporting of an isopod enclosure via a Raspberry Pi.

## Installation

```
git clone https://github.com/esse-est/isopi.git
```

```bash
pip install -r isopi/requirements.txt
```

## Usage

```python
python3 main.py
```
